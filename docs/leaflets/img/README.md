# Leaflet screenshots

`impactasset.html` (the Solution column) loads three phone screenshots from this
folder. Save the screenshots here with these **exact** filenames:

| File | Screenshot to use |
|------|-------------------|
| `plan-next-week.png` | The "Plan next week" screen (pledge days + Your Stake) |
| `group-pot.png`      | The Group tab "Weekly Pot" + member breakdown screen |
| `gymspace.png`       | The Progress / pixel-art GymSpace screen |

Tall phone screenshots (≈9:19.5) work best — they're cropped `object-fit: cover`
from the top. PNG or JPG both fine; just keep the filename the same.
